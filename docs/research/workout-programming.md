## Volume Landmarks (RP Framework - Mike Israetel)

| Landmark | Definition | Purpose |
|----------|-----------|---------|
| **MV** (Maintenance Volume) | Minimum sets to maintain gains | Deload weeks, busy periods |
| **MEV** (Minimum Effective Volume) | Minimum sets to make progress | Beginners, recovery-limited |
| **MAV** (Maximum Adaptive Volume) | Sweet spot for most gains | Where most training should be |
| **MRV** (Maximum Recoverable Volume) | Most you can recover from | Overreaching blocks only |

### Weekly Sets Per Muscle Group

| Muscle Group | MV | MEV | MAV | MRV |
|-------------|-----|-----|-----|-----|

## Progressive Overload: Double Progression

The primary progression model for FitCoach:

```
Target: 3x8-12 reps

Week 1: 60kg x 8, 8, 7     → Stay at 60kg
Week 2: 60kg x 10, 9, 8    → Stay at 60kg
Week 3: 60kg x 12, 11, 10  → Stay at 60kg
Week 4: 60kg x 12, 12, 12  → HIT TOP! Next session: 62.5kg x 8+
```

**Rules:**
1. When ALL sets hit the top of the rep range → increase weight
2. Weight increase: 2.5kg upper body, 5kg lower body (1.25kg/2.5kg for smaller muscles)
3. After increasing weight, reps will drop — this is expected
4. If reps drop below the bottom of the range for 2+ sessions → deload

## Deload Protocol

**When:** Every 4-6 weeks, or when performance stalls 2+ sessions
**How:**
- Reduce volume by 40-50% (cut sets, keep reps same)
- Keep weight the same or reduce by 10%
- Duration: 1 week
- Purpose: Dissipate accumulated fatigue, allow supercompensation

**Auto-detection signals (for app):**
- RPE consistently 9-10 on working sets
- Weight/reps declining for 2+ consecutive sessions
- User reports poor sleep, high stress, joint pain

## Rep Range Guidelines by Goal

| Goal | Rep Range | Rest Period | RIR (Reps in Reserve) |
|------|-----------|-------------|----------------------|
| Strength | 3-6 | 3-5 min | 1-2 |
| Hypertrophy | 6-12 | 1.5-3 min | 1-3 |
| Muscular Endurance | 12-20 | 30-90s | 1-2 |
| Strength-Hypertrophy Mix | 6-10 | 2-3 min | 1-2 |

**Note:** Schoenfeld 2017 showed hypertrophy occurs across ALL rep ranges (even 25-35 reps) as long as sets are taken close to failure. The 6-12 range is simply most time-efficient.

## Beginner vs Intermediate Progression

### Beginner (0-12 months training)
- **Volume:** Start at MEV, increase by 1-2 sets/week per muscle group every 2-3 weeks
- **Frequency:** 2-3x/week full body recommended
- **Progression:** Linear — add weight every session when rep target hit
- **Focus:** Learning movement patterns, mind-muscle connection
- **RPE target:** 6-7 (3-4 RIR) — learn form before pushing intensity

### Intermediate (1-3 years training)
- **Volume:** Start at lower end of MAV, periodize up to upper MAV
- **Frequency:** 4-6x/week (U/L or PPL)
- **Progression:** Double progression — add reps first, then weight
- **Focus:** Progressive overload, weak point identification
- **RPE target:** 7-9 (1-3 RIR)

## Algorithm: Split Selection Logic

```
IF training_days <= 3:
    recommend = "Full Body"

ELIF training_days == 4:
    recommend = "Upper/Lower"

ELIF training_days >= 5:
    IF experience < 6_months:
        recommend = "Upper/Lower" (4 days) + optional cardio days
        note = "PPL requires more recovery capacity"
    ELSE:
        recommend = "Push/Pull/Legs"

// User can always override recommendation
```

## Algorithm: Volume Progression (Mesocycle)

```
Week 1: MEV + 2 sets (introductory)
Week 2: MEV + 4 sets
Week 3: MAV low end
Week 4: MAV mid
Week 5: MAV high end (approaching MRV)
Week 6: DELOAD (back to MV)

// Then start new mesocycle at Week 1 volume + slight increase
```

## TDEE Calculation

### Mifflin-St Jeor (no body fat % known)
```
Male:   BMR = 10 × weight(kg) + 6.25 × height(cm) - 5 × age - 5  (was incorrectly +5)
Female: BMR = 10 × weight(kg) + 6.25 × height(cm) - 5 × age - 161

TDEE = BMR × Activity Multiplier
```

### Katch-McArdle (body fat % known)
```
LBM = weight × (1 - bodyfat%)
BMR = 370 + 21.6 × LBM(kg)

TDEE = BMR × Activity Multiplier
```

### Activity Multipliers
| Level | Multiplier | Description |
|-------|-----------|-------------|
| Sedentary | 1.2 | Desk job, no exercise |
| Light | 1.375 | 1-3 days/week |
| Moderate | 1.55 | 3-5 days/week |
| Active | 1.725 | 6-7 days/week |
| Very Active | 1.9 | Physical job + daily training |

### Goal Adjustments
| Goal | Calorie Adjustment |
|------|-------------------|
| Fat Loss (moderate) | TDEE - 300-500 kcal |
| Fat Loss (aggressive) | TDEE - 500-750 kcal |
| Muscle Gain (lean bulk) | TDEE + 200-300 kcal |
| Muscle Gain (standard) | TDEE + 300-500 kcal |
| Maintenance | TDEE ± 0 |

### Macro Split
| Goal | Protein | Fat | Carbs |
|------|---------|-----|-------|
| Fat Loss | 2.0-2.4 g/kg | 0.8-1.0 g/kg | Remainder |
| Muscle Gain | 1.6-2.2 g/kg | 0.8-1.0 g/kg | Remainder |
| Maintenance | 1.6-2.0 g/kg | 0.8-1.0 g/kg | Remainder |

**Protein priority:** Always calculated first. Evidence (Helms 2014): 1.6-2.2 g/kg is sufficient for maximizing muscle protein synthesis. Higher end during caloric deficit.
